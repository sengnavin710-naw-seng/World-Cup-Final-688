# Team Logo Folder

Place each team logo file in this folder.

Use this naming format:

- `arg.png`
- `bra.png`
- `fra.png`
- `ger.png`

The app will look for each logo by the team code in lowercase:

- `ARG` -> `/team-logos/arg.png`
- `BRA` -> `/team-logos/bra.png`
- `JPN` -> `/team-logos/jpn.png`

Recommended:

- square image
- PNG format
- transparent background if possible
